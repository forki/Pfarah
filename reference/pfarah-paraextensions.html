collect prop obj
Signature: prop:string -> obj:ParaValue -> ParaValue []

Finds all the properties of the object with a given key and aggregates all the values under a single array.
findOptional objs
Signature: objs:seq<ParaValue> -> seq<string * bool>

Given a sequence of similar objects, return sequence of tuples where it is the name of the property and a boolean value denoting whether the property did not occur in all given objects.

Example Return Value

[("hello", true); ("world", false)]

means that the "hello" property is optional and the "world" property is required in each object

stringDefault _arg1

tryFind prop obj
Signature: prop:string -> obj:ParaValue -> ParaValue option

Tries to find the first property of the object that has the given key. If a property is found then Some ParaValue will be returned else None
